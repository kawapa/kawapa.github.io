# Skróty klawiszowe VSC, które chciałbym zapamiętać
{: .no_toc }

---

## Ogólne

| Skrót klawiszowy                 | Znaczenie                                           |
| :------------------------------- | :-------------------------------------------------- |
| Ctrl + Shift + P                 | Panel komend                                        |
| F1                               | Panel komend                                        |
| Ctrl + B                         | Pokaż / ukryj sidebar                               |
| Ctrl + K + Z                     | **Zen Mode**                                        |
| Ctrl + H                         | **Znajdz i zamień**                                 |
| F3                               | Znajdź następny                                     |
| Ctrl + P                         | Otwórz plik                                         |
| Ctrl + K + F                     | Zamknij bieżący folder                              |
| Ctrl + \                         | **Otwórz bieżący plik w kolumnie obok**             |
| Ctrl + N                         | Nowy plik                                           |
| Ctrl + W                         | **Zamknij aktywny plik / kartę**                    |
| Ctrl + K + Ctrl + W              | Zamknij wszystkie pliki                             |
| Ctrl + Shift + T                 | Przywróc wszystkie zamknięte ostatnio pliki / karty |
| Ctrl + Tab                       | Przełącz się na następny plik / kartę               |
| Ctrl + Tab (nie puszczając Ctrl) | Przejdź do wybranego pliku / karty                  |

---

## Edycja kodu

| Skrót klawiszowy           | Znaczenie                                                |
| :------------------------- | :------------------------------------------------------- |
| Ctrl + X (bez zaznaczenia) | Wytnij aktywny wiersz                                    |
| Ctrl + C (bez zaznaczenia) | Kopiuj aktywny wiersz                                    |
| Ctrl + Enter               | **Wstaw wiersz poniżej**                                 |
| Ctrl + Shift + Enter       | **Wstaw nowy wiersz powyżej**                            |
| Ctrl + ] / Ctrl + [        | **Powiększ / pomniejsz wcięcie**                         |
| Ctrl + Home/End            | Przejdź do początku / końca pliku                        |
| F12                        | **Idź do definicji zmiennej / funkcji**                  |
| Ctrl + Shift + F10         | **Pokaż definicję (w małym okienku)**                    |
| Shift + F12                | Pokaż wszystkie odwołania                                |
| F2                         | **Zmień nazwę symbolu**                                  |
| Alt + click                | Wstaw kursor (do pisania w wielu miejscach jednocześnie) |
| Shift + Alt + ↑ / ↓        | Wstaw kursora powyżej / poniżej                          |

## Debug

| Skrót klawiszowy | Znaczenie          |
| :--------------- | :----------------- |
| F9               | Toggle breakpoint  |
| F5               | Rozpocznij / Dalej |
| F11 / Shift+F11  | Step into/out      |
| F10              | Step over          |
| Shift+F5         | **Zatrzymaj**      |
| Ctrl+K Ctrl+I    | Show hover         |
